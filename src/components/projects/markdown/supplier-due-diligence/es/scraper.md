# Risk Entity Scraper

Servicio de extracción y análisis de entidades desarrollado con **TypeScript + Puppeteer**, encargado de automatizar la consulta de listas de alto riesgo como parte del proceso de **screening de proveedores** dentro de la plataforma Supplier Due Diligence.

El servicio permite obtener información desde fuentes externas, procesar los resultados y exponerlos de forma estructurada para su consumo por el backend principal.

## Valor para el negocio

Automatiza la búsqueda de posibles coincidencias en listas de riesgo, reduciendo procesos manuales de investigación y permitiendo evaluaciones más rápidas y consistentes durante la validación de proveedores.

## Aspectos técnicos destacados

- Servicio de scraping desarrollado con TypeScript.
- Automatización de navegación web mediante Puppeteer.
- Extracción y procesamiento de información desde fuentes externas.
- Arquitectura modular orientada a servicios.
- Endpoints internos protegidos mediante autenticación entre servicios.
- Comunicación segura con backend mediante claves internas.
- Configuración flexible mediante variables de entorno.
