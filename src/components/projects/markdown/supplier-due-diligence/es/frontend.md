# Frontend - Supplier Due Diligence

Aplicación SPA desarrollada con **React + TypeScript** para la gestión y evaluación de proveedores dentro de un proceso de **Debida Diligencia (Due Diligence)**.

La plataforma permite centralizar información de proveedores y ejecutar procesos de **screening**, realizando la comparación automática contra listas de alto riesgo para identificar posibles coincidencias o señales de alerta antes de establecer relaciones comerciales.

## Valor para el negocio

Ayuda a las organizaciones a reducir revisiones manuales, mejorar la trazabilidad de evaluaciones y facilitar la toma de decisiones en procesos de cumplimiento, gestión de proveedores y prevención de riesgos.

## Aspectos técnicos destacados

- React + TypeScript como SPA moderna.
- Arquitectura modular orientada a funcionalidades.
- Componentes reutilizables y escalables.
- Integración mediante APIs REST con backend .NET.
- Autenticación segura mediante **JWT utilizando cookies HttpOnly y Secure para comunicación HTTPS**.
- Validaciones de formularios y experiencia de usuario enfocada en procesos empresariales.
