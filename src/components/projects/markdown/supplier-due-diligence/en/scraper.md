# Risk Entity Scraper

Extraction and entity analysis service developed with **TypeScript + Puppeteer**, responsible for automating high-risk list queries as part of the **supplier screening** process within the Supplier Due Diligence platform.

The service retrieves information from external sources, processes the results, and exposes structured data for consumption by the main backend service.

## Business Value

Automates the search for potential matches across risk lists, reducing manual investigation efforts and enabling faster and more consistent supplier evaluations during the validation process.

## Technical Highlights

- Web scraping service developed with TypeScript.
- Browser automation using Puppeteer.
- Extraction and processing of information from external sources.
- Modular service-oriented architecture.
- Internal endpoints protected through service-to-service authentication.
- Secure communication with backend services using internal keys.
- Flexible configuration through environment variables.
