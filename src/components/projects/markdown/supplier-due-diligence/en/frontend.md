# Frontend - Supplier Due Diligence

SPA application developed with **React + TypeScript** for supplier management and evaluation within a **Due Diligence** process.

The platform centralizes supplier information and executes **screening processes**, performing automated comparisons against high-risk lists to identify potential matches or risk indicators before establishing business relationships.

## Business Value

Helps organizations reduce manual review processes, improve evaluation traceability, and support decision-making in compliance, supplier management, and risk prevention workflows.

## Technical Highlights

- Modern SPA built with React + TypeScript.
- Modular architecture organized by business functionality.
- Reusable and scalable components.
- REST API integration with a .NET backend.
- Secure authentication using **JWT with HttpOnly and Secure cookies over HTTPS**.
- Form validations and user experience focused on enterprise workflows.
